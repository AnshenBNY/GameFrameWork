```json
}
```

在 PowerShell 里用 `Get-Command node | Select-Object -ExpandProperty Source` 查你的实际路径，替换上面的路径。

脚本依赖 stdin 里的 `transcript_path`，或环境变量 **`CURSOR_TRANSCRIPT_PATH`**（文档：仅在启用 transcripts 时提供）。
